# プロダクト概要

FootballContractSim は、サッカー選手の契約や移籍、市場価値を扱う「契約シミュレーションゲーム」です。現実のフットボールクラブ運営に近い意思決定（スカウティング、契約更改、移籍交渉など）をブラウザ上で体験できることを目的としています。

## コア機能パターン

- 選手データ管理：`Player` ドメインを中心に、ポジション・年齢・能力値・市場価値・契約期間などの属性を一貫したスキーマで扱う
- 契約・評価ロジック：選手の能力値やポテンシャル、年齢などから契約条件や市場価値を評価するロジックを段階的に拡張できるように設計する
- 一覧・フォーム UI：Next.js App Router 上で、一覧テーブルと作成フォームを組み合わせた管理画面スタイルの UI を提供する
- シミュレーション拡張性：将来的な「シーズン進行」「クラブ財政」「リーグ」などの概念を追加しやすいよう、ドメインを分離しておく

## 想定ユースケース

- フットボールゲームやスポーツマネジメントゲームのプロトタイプとして、契約・移籍周りの体験を検証する
- サッカー好きの個人開発者が、選手データや契約アルゴリズムを試行錯誤するための遊び場・実験場として使う
- 将来の機能追加（リーグ戦シミュレーション、監督・戦術、財務管理など）の土台となる「選手と契約」の最小コアを固める

## 提供価値パターン

- ゲーム寄り／業務寄りのどちらにも広げられる「契約ドメインの最小実装」を共有できる
- Next.js + Prisma + MySQL を前提とした、モダンなフルスタック構成でのスポーツシミュレーション実装例になる
- ドメインを分離しやすい構造により、将来的に AI や高度なアルゴリズムを組み込みやすい
